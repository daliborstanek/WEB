# Escape from Tarkov

## **Žák**
Dalibor Staněk

## **Stručný popis tématu (3–5 vět)**

Webová prezentace zaměřená na hru Escape from Tarkov, která hráčům přiblíží její hlavní mechaniky, frakce, mapy a základní principy přežití. Stránka bude mít podobu one-page webu s několika sekcemi, jejichž obsah bude dynamicky načítán z JSON souborů pomocí AJAXu. Cílem je vytvořit moderní, přehlednou a vizuálně atraktivní prezentaci, která shrnuje klíčové informace pro nové i pokročilejší hráče. Web se zaměří na strukturované předání dat, jednoduché animační prvky a responzivní zpracování.

## **Cílová skupina**

Noví hráči hledající základní orientaci ve hře, fanoušci hry Escape from Tarkov a lidé, kteří se zajímají o realistické FPS survival tituly.

## **Návrh obsahových sekcí**

1. Stručné představení Escape from Tarkov, jejího žánru a vývojářů.
    
2. Informace o PMC frakcích USEC a BEAR + popis systému Scav.
    
3. Dynamicky načítaný přehled herních map s krátkými popisy, doporučeními a obtížností.
    
4. Výstroj, health systém, extrakce, inventář, lootování.
    
5. Kontaktní formulář + odkazy na oficiální zdroje, wiki nebo komunitu.
    
## **Návrh designu / wireframe**

- Temnější herní tématika (odstíny šedé, černé, khaki, zelené), kontrastní akcentní barvy.
    
- Přehledné blokové uspořádání, sekce oddělené vizuálními prvky a ikonami.
    
- Karty map nebo frakcí zobrazené v gridu (2–3 sloupce podle šířky obrazovky).
    
- Moderní UI s použitím prvků jako glassmorphism, shadow a responsivních Tailwind komponent.
    

## **Použité technologie**

- **Frontend:** HTML, CSS nebo Tailwind CSS, JavaScript
    
- **Data:** JSON soubory pro čtyři textové sekce + JSON nabídka štěňat
    
- **AJAX:** načítání obsahu sekcí a seznamu štěňat
    
- **Backend:** jednoduchý PHP endpoint pro odeslání kontaktního formuláře
    
- **Verzování:** Git + GitHub (repozitář dostupný vyučujícímu)
